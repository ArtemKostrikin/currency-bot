# Currency Bot

Telegram-бот для конвертации валют через API.

## Функционал
- Конвертация валют по актуальному курсу
- Поддержка RUB, USD, EUR
- Валидация ввода
- Обработка ошибок API

## Стек
- Python
- pyTelegramBotAPI
- exchangerate.host API

## Запуск

```bash
pip install -r requirements.txt
python bot.py
